# **Common First-Time Issues**

### 1. Black Screen on Start
- **Solution: Change Graphics Controller in Settings > Display.**

### 2. Stuck at Apple Logo
- **Solution: Be patient, first boot can take 10-15 minutes.**

### 3. Mouse Not Working
- **Solution: Settings > System > Pointing Device > USB Tablet.**

### 4. No Internet
- **Solution: Settings > Network > Adapter 1 > Attached to: NAT.**
